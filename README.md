# Como executar a aplicação

Primeiramente é necessário executar a aplicação backend, conforme instruido no repositório correspondente.

Logo em seguida, dentro da pasta *extrato* execute:

### `npm install`

Logo em seguida execute

### `npm start`

## A aplicação

A primeira tela pedirá o número da conta (id). Deve-se informar o id e clicar no botão de entrar.

A partir disso, será exibida a tela de extrato e podem ser aplicados os filtros pertinentes.

A paginação considera itens por página, então os botões só ficam ativos a partir de 5 itens.
